## Develop It

Watch for changes and build both app.js and component.js.
```
npm run dev-component & npm run dev-app &
```

Somewhat standard on running tests, except uses 
`make clean && make --jobs test`
and builds temporary `*.tmp.js` files.
```
npm run test
```
